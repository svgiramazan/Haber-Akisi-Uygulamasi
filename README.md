# Haber Akışı Uygulaması

Bu proje, **NewsAPI** kullanarak gerçek zamanlı haberleri listeleyen, kategorilere ayıran ve kullanıcıların beğendikleri haberleri favorilerine eklemesine olanak tanıyan bir Flutter mobil uygulamasıdır.

##  Özellikler

* **Anlık Haber Takibi:** En güncel haberleri farklı kategorilerde (Teknoloji, Spor, Ekonomi, Sağlık vb.) görüntüleme.
* **Dinamik Arama:** Anahtar kelime kullanarak haberler içerisinde arama yapabilme.
* **Favori Sistemi:** Beğenilen haberleri yıldız ikonuna basarak favorilere ekleme ve uygulama kapatılsa dahi bu verileri `shared_preferences` ile saklama.
* **Haber Detayı ve Paylaşma:** Haberin içeriğini okuma ve `share_plus` paketi ile sosyal medyada paylaşma.
* **WebView Entegrasyonu:** Haberin orijinal kaynağına uygulama içerisinden veya tarayıcıdan hızlı erişim.

##  Kullanılan Teknolojiler

* **Flutter & Dart**
* **http:** API istekleri için.
* **shared_preferences:** Yerel veri depolama için.
* **share_plus:** Sosyal paylaşım desteği için.
* **flutter_dotenv:** API anahtarı güvenliği için.
